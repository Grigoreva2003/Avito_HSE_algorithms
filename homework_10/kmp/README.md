# Knuth–Morris–Pratt

## Задача
Реализовать поиск подстроки в строке с помощью алгоритма Кнута — Морриса — Пратта 

К решению нужно прикрепить отчет с

* объяснением алгоритма
* оценкой сложности итогового алгоритма

Тесты продолжаем писать.

## Как работает КМП
1) Строим префикс-функцию ($\text{pi}$) для шаблона за O(m): для каждой позиции храним длину наибольшего собственого префикса, совпадающего с суффиксом.
2) Идём по тексту и шаблону. При несовпадении не откатываем текст, а двигаем позицию в шаблоне по $\text{pi}$, тем самым «прыгая» к следующему возможному префиксу. При совпадении увеличиваем текущую длину совпавшего префикса.
3) Когда длина совпавшего префикса достигает m, фиксируем вхождение и продолжаем поиск, используя $\text{pi}$ для следующего возможного совпадения.

## Сложность
* Время: O(n + m), где n — длина текста, m — шаблона ($\text{pi}$ за O(m) + поиск за O(n)).
* Память: O(m) для хранения $\text{pi}$-массива.
